# Glossary

- **Business**: An organization you interact with (prospect/customer/partner).
- **Contact**: A person. A contact can be linked to multiple businesses.
- **External user**: A local Identity user account (customer/partner) not in Entra.
- **Staff user**: An Entra-authenticated user (internal employee).
- **Customer Status**: The stage/status of a *relationship* to a Project via `ProjectParty` (later). “Potential lead” is a status value here.
- **Feature Permission**: Permission key controlling access to a screen/action.
- **Record-level access**: Rules restricting which records a user can see/edit (assignment/manager).
- **Assignment**: An explicit link between a user and a record (business/contact/project), used for record-level access checks.
- **StatusType**: Category of statuses (BusinessStatus, ProjectStatus, LeaveRequestStatus, etc.).
- **Tenant**: Logical partition for future multi-tenant capability.
