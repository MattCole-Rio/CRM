# CRM Design Pack

This folder contains the initial design pack for the CRM system.

## Contents
- `architecture.md` — Architecture overview, layering, module boundaries
- `domain-model.md` — Domain model (aggregates, entities, invariants)
- `database-schema.md` — SQL/EF-oriented schema outline (tables, keys, relationships)
- `rbac-permissions.md` — RBAC model: roles, permissions, overrides, record-level access
- `auth-identity.md` — Entra ID + local Identity approach
- `ui-navigation.md` — Blazor Server UI structure and navigation
- `status-model.md` — Configurable status pipelines
- `reckon-csv-integration.md` — Reckon Accounts Premier CSV import/export strategy (no sample files)
- `testing-strategy.md` — Unit/integration test plan
- `backlog-issues.md` — Ready-to-paste GitHub issue backlog (epics + stories)
- `glossary.md` — Terminology used in this project
